MTK
===========================
	# CyanogenMod 13.0 for mt6592 (Fly Tornado One)

* Working
  * RIL 
  * Dual SIM 
  * Audio
  * Sensors
  * Wifi
  * MTP, ADB etc.
  
* Not Working
  * Bluetooth
  * Camera
  * GPS
 
* Sync CyanogenMod source:

        # repo init -u git://github.com/ferhung-mtk/android.git -b cm-13.0        
        # repo sync


* Compilation
       
        # source build/envsetup.sh    
        # brunch cm_IQ4511-userdebug
